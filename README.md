|---------|------------|---------------|
| Fake deliveries | Weak GPS validation | Geotag + photo protocol |
| Stuck refunds | Manual processing | 24h auto-refund API |
| Poor support | Low-authority agents | AI triage + escalation paths |
| Misleading UX | Confusing checkout | Consent-first redesign |
| High costs | Inefficient routing | Dynamic rider pricing |
